# 📚 Инструкция по установке OCR для RAG-сервиса на Windows

## Требования
- Windows 10/11
- Python 3.10+
- 8 ГБ ОЗУ (оптимизировано)

---

## 🔧 Шаг 1: Установка Poppler (для pdf2image)

**Poppler** — это библиотека для работы с PDF, необходимая для конвертации страниц PDF в изображения.

### Вариант А: Через установщик (рекомендуется)

1. Скачайте последнюю версию Poppler для Windows:
   - https://github.com/oschwartz10612/poppler-windows/releases/
   - Выберите файл `Release-24.08.0-0.zip` (или новее)

2. Распакуйте архив в удобную директорию, например:
   ```
   C:\Program Files\poppler
   ```

3. Добавьте путь к бинарникам в переменную окружения PATH:
   - Откройте **Панель управления** → **Система** → **Дополнительные параметры системы**
   - Нажмите **Переменные среды**
   - В разделе **Системные переменные** найдите `Path`
   - Нажмите **Изменить** → **Создать**
   - Добавьте путь: `C:\Program Files\poppler\Library\bin`
   - Нажмите **OK** во всех окнах

4. Проверьте установку:
   ```cmd
   pdftoppm -h
   ```
   Должна отобразиться справка.

### Вариант Б: Через pip (альтернативный)

```bash
pip install python-poppler
```

Но этот вариант может требовать дополнительных настроек.

## 🔧 Шаг 2: Установка Tesseract OCR

**Tesseract** — движок оптического распознавания символов.

1. Скачайте установщик Tesseract для Windows:
   - https://github.com/UB-Mannheim/tesseract/wiki
   - Рекомендуется версия **tesseract-ocr-w64-setup-5.x.x.exe**

2. Запустите установщик:
   - Путь установки: `C:\Program Files\Tesseract-OCR`
   - ✅ Отметьте **Additional language data**: **Russian** и **English**
   - Завершите установку

3. Добавьте Tesseract в PATH:
   - Откройте **Панель управления** → **Система** → **Дополнительные параметры системы**
   - Нажмите **Переменные среды**
   - В разделе **Системные переменные** найдите `Path`
   - Нажмите **Изменить** → **Создать**
   - Добавьте путь: `C:\Program Files\Tesseract-OCR`
   - Нажмите **OK** во всех окнах

4. Проверьте установку:
   ```cmd
   tesseract --version
   ```
   Должна отобразиться версия Tesseract.

## 🔧 Шаг 3: Установка зависимостей Python

1. Перейдите в директорию проекта:
   ```cmd
   cd C:\path\to\rag-service
   ```

2. Установите зависимости:
   ```cmd
   pip install -r requirements.txt
   ```

   Будут установлены:
   - `PyMuPDF>=1.23.21` — для извлечения текста из PDF
   - `pytesseract>=0.3.10` — Python-обёртка для Tesseract
   - `pdf2image>=1.16.0` — конвертация PDF в изображения
   - `Pillow>=9.0.0` — работа с изображениями

## 🔧 Шаг 4: Проверка работы

1. Создайте тестовый скрипт `test_ocr.py`:

```python
from app.document_processor import document_processor

# Путь к тестовому PDF (отсканированному учебнику)
pdf_path = "data/textbooks/programming/test_scanned.pdf"

# Извлечение текста
pages = document_processor.extract_text_from_pdf(pdf_path)

print(f"Найдено страниц: {len(pages)}")
for text, page_num in pages[:3]:  # Первые 3 страницы
    print(f"\n=== Страница {page_num} ===")
    print(text[:200])  # Первые 200 символов
```

2. Запустите тест:
   ```cmd
   python test_ocr.py
   ```

3. Проверьте логи:
   - Если видите `Мало текста на странице X, запускаю OCR...` — OCR активирован
   - Если видите `OCR распознал текст на странице X` — распознавание успешно

## 🔧 Шаг 5: Запуск сервиса

1. Настройте `.env` файл (если ещё не настроен):
   ```env
   LLM_PROVIDER=ollama
   EMBEDDING_FUNCTION=onnx_minilm_l6_v2
   OLLAMA_BASE_URL=http://localhost:11434
   OLLAMA_MODEL=qwen2.5-3b-instruct-q4km
   CHUNK_SIZE=500
   CHUNK_OVERLAP=50
   MAX_QUERY_LENGTH=1000
   MAX_CONTEXT_DOCUMENTS=5
   ```

2. Запустите сервис:
   ```cmd
   python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
   ```

3. Откройте браузер:
   - API Docs: http://127.0.0.1:8000/docs
   - Health check: http://127.0.0.1:8000/health

## ⚡ Оптимизация для 8 ГБ ОЗУ

Сервис оптимизирован для работы на слабых ноутбуках:

1. **Обработка по одной странице** — не загружает все страницы в память одновременно
2. **DPI = 150** — баланс между качеством OCR и потреблением памяти
3. **Временная директория** — изображения удаляются сразу после обработки
4. **PyMuPDF вместо PyPDF2** — быстрее и меньше памяти

### Рекомендации:
- Закройте лишние приложения при обработке больших учебников
- Обрабатывайте учебники по одному предмету за раз
- Для очень больших PDF (>500 страниц) делайте перерывы

## 🐛 Решение проблем

### Ошибка: `pdf2image.exceptions.PDFInfoNotInstalledError`
**Решение:** Poppler не найден в PATH. Проверьте шаг 1.

### Ошибка: `TesseractNotFoundError`
**Решение:** Tesseract не найден в PATH. Проверьте шаг 2.

### Ошибка: `DataNotFoundError` для языка `rus`
**Решение:** При установке Tesseract не выбран русский язык. Переустановите Tesseract с языками.

### OCR возвращает пустой текст
**Возможные причины:**
- Низкое качество скана (размыто, тёмное)
- Нестандартный шрифт или рукописный текст
- Слишком маленький DPI (попробуйте увеличить до 200 в коде)

### Медленная обработка
**Нормально!** OCR — ресурсоёмкая операция.
- 1 страница ≈ 2-5 секунд на среднем ноутбуке
- Учебник 100 страниц ≈ 3-8 минут
